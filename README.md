# Weather App with Streamlit and OpenWeatherMap

This is a simple weather application built using Streamlit and the OpenWeatherMap API. It allows users to fetch and display weather information for any city.

## Prerequisites
- Python 3.7+
- Streamlit
- Requests library
- OpenWeatherMap API Key

## Installation
1. Clone the repository or download the files.
2. Install the required packages:
   ```bash
pip install -r requirements.txt
```
3. Obtain an API key from [OpenWeatherMap](https://home.openweathermap.org/users/sign_up).

## Running the App
1. Run the Streamlit app:
   ```bash
streamlit run app.py
```
2. Enter your OpenWeatherMap API Key and the city name in the app.
3. Click "Get Weather" to see the weather details.

## Notes
- Ensure your API key is kept secure and not exposed publicly.
